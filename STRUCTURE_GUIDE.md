## Responsive Layout Changes

### Desktop View (> 768px)
```
Navigation: Horizontal menu items side by side
Hero: 2 columns (text left, image right)
About: 2 columns (text left, stats right)
Experience: Timeline with vertical line
Skills: 4 columns (all categories visible)
Projects: 4 columns grid (4 projects in 2 rows)
Certifications: 4 columns grid
Research: Full width with icons on left
Interests: 4 columns grid
Contact: 3 columns (email, linkedin, location)
```

### Tablet View (480-768px)
```
Navigation: Responsive menu
Hero: Stacked (text top, image bottom)
About: Stacked (text top, stats bottom)
Experience: Timeline adjusted
Skills: 2 columns (alternating)
Projects: 2 columns grid
Certifications: 2 columns grid
Research: 1 column with left border
Interests: 2 columns grid
Contact: 3 columns or stacked
```

### Mobile View (< 480px)
```
Navigation: Hamburger menu
Hero: Full width, stacked
About: Single column
Experience: Simplified timeline
Skills: 1 column, cards stack
Projects: 1 column cards
Certifications: 1 column cards
Research: Single column
Interests: Single column
Contact: Stacked vertically
Footer: Centered, stacked
```

## Font Hierarchy

```
Main Logo: 1.5rem, Bold (700)
Hero Title: 3rem (Desktop), 2rem (Mobile), Bold (700)
Section Title: 2.5rem (Desktop), 1.5rem (Mobile), Bold (700)
Card Titles: 1.3rem, Color varies (600-700 weight)
Labels: 1.1rem, Semi-bold (600)
Body Text: 1rem-1.05rem, Regular (400)
Small Text: 0.9rem-0.95rem, Regular (400)
Captions: 0.85rem, Regular (400)
```

```

Card Padding: 25-30px
Component Gap: 20-30px
```

## Interactive Elements

### Buttons
- Primary: Blue with gradient
- Secondary: Transparent with blue border
- Hover: Transform up, shadow increase
- Active: Color change
- Focus: Outline for accessibility

### Cards
- Default: Light shadow
- Hover: Larger shadow, slight up translation
- Active: Border highlight or color change

### Navigation
- Default: Text color
- Hover: Underline animation, color change
- Active: Full underline, bright color

### Icons
- Size: 1rem to 3rem depending on context
- Color: Matches text or accent color
- Hover: Scale up, color change

---

## Animation Timeline

```
On Load:
- Profile picture: Fade in from right
- Text: Fade in normally

On Scroll:
- Cards: Fade in from bottom with delay
- Numbers: Count animation
- Text: Fade in as visible

On Hover:
- Cards: Translate up 5-10px
- Buttons: Translate up 2px
- Links: Underline appears
- Icons: Scale 1.05x

On Interaction:
- Mobile menu: Slide in/out
- Image slider: Fade transition
- Smooth scroll: 1000ms duration
```

---

## Accessibility Features

```
Typography:
- Sufficient color contrast (>4.5:1)
- Readable font sizes (16px minimum)
- Line height: 1.6 for readability
- Proper heading hierarchy

Navigation:
- Keyboard navigation supported
- Tab order logical
- Focus indicators visible
- Skip links for main content

Images:
- Alt text provided
- Decorative images marked
- Text not embedded in images
- Icon fonts have labels

Forms (if added):
- Labels associated with inputs
- Error messages clear
- Required fields marked
- Instructions provided
```
